html is a standard markup language for creating webpages and it describes the structure of a webpage

The timeline of html's historical development

In 1991 Tim Berners invented html In 1995 html working group defined html 2.0 In 1997 W3C recommended html 3.2 In 1999 W3C recommended html 4.01 In 2000 W3C recommended html 1.0 In 2017 W3C recommended html 5.2
